# VS Code 단축키

자주 사용하는 VS Code 단축키 정리

## 기본 명령어

- **들여쓰기** : `ctrl + [`
- **내어쓰기** : `ctrl + ]`
- **통합 터미널 열기** : `ctrl + j`

### 찾기

- **찾기** : `ctrl + f`
- **해당 단어 전체 선택** `ctrl + f`
- **선택하며 찾기** : `ctrl + d`
- **프로젝트 전체에서 찾기** : `ctrl + shift + f`

### 이동

- **파일 시작으로 가기** : `ctrl + home`
- **파일 끝으로 가기** : `ctrl + end`
- **퀵 오픈** : `ctrl + p`

### 삭제

- **현재 줄 지우기** : `shift + ctrl + k`

✌️ 유용한 단축키를 알게 되면 계속해서 추가 ✌️
