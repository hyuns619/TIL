## 조건문 문법

> 조건문이란 표현식이 참으로 평가될 때, 실행되는 블록이다.

```js
if (true) {
	console.log('항상 실행이 됩니다.');
} // 표현식이 참이면 실행된다.

if (false) {
	console.log('실행되지 않습니다');
} // 표현식이 거짓이면 실행되지 않는다.
```

**Falsy 값**

> Boolean으로 평가될 때, 거짓인 값이다.

- false, 0, ", null, undefined, NaN

**Truethy 값**

> Boolean으로 평가될 때, 참인 값이다.

- true, 37, 'Mark', {}, []
